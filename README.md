## Projects

### Coin Flip Simulation

An interactive web application that visualizes the probability distribution of random walks through coin flip simulations.

**Features:**
- Additive and multiplicative game modes
- Configurable number of simulations and flips per game
- Real-time statistical analysis (average, median, min, max)
- Interactive histogram visualization with Chart.js
- Logarithmic scale for multiplicative mode

## Getting Started

Each project is self-contained in its own directory. Simply open the `index.html` file in your browser to run any simulation.

```bash
# Clone the repository
git clone https://github.com/claudiofahey/vibetoys.git

# Navigate to a project
cd vibetoys/coinflip

# Open in browser
open index.html
```
